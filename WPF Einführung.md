# WPF - Windows Presentation Foundation

## Einführung

+ Neue Technologie (seit 2006 :-) mit .NET 3.0) zur Entwicklung von Windowsanwendungen
+ Strikte Trennung von Präsentations- und Geschäftslogik
	- Mittels Oberflächenbeschreibungssprache XAML (eXtensible Application Markup Language)
+ Grundlegende Anwendungen
	- Windows Desktop Anwendungen
	- Windows Store-Apps (ab Windows 8 mittels WinRT)
